
=== DHD Lotto Application ===
Contributors: Dirk Harriman

== Description ==

This is a simple javascript based application that allows the user to pick lotto numbers. It was originally intended to be used in an online application. There was another backend piece to the application that handled users and stored picks. The daily picks would then be printed onto lotto forms to be fed into the lotto machine. The project ran into several hitches. First the lottery commissions would not allow a company to work as a lottery broker. Second, we couldn't find an off the shelf printer that would accurately hold and print the lotto forms. The work was chalanging as I'd never created a print appliction like this before. It was written in C# as a .Net desktop application. I also wrote another version using MS Silverlight (since depreciated)
